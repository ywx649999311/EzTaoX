# EzTao*X* Example Notebooks

In this directory, you will find notebooks that walk you through the basics of 
using EzTaoX.

Notebooks in this folder:

- `01_DRW.ipynb`: single-band DRW fitting
- `02_DHO.ipynb`: single-band DHO/CARMA(2,1) fitting
- `03_MultibandFitting.ipynb`: multiband fitting with shared latent kernels
- `04_MultibandSimulation.ipynb`: multiband light-curve simulation with `MultiVarSim`

Additional notebooks elsewhere in the docs tree:

- `../pre_executed/04_TransferFunctions.ipynb`: transfer-function convolution and direct dense fitting
- `../pre_executed/experimental/05_KernelApproximations.ipynb`: experimental SHO-series approximations to dense convolved kernels

For more user information, see https://eztaox.readthedocs.io/en/latest/index.html

For EzTaoX source code, see https://github.com/LSST-AGN-Variability/EzTaoX
